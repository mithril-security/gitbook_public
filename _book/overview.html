Overview

Our solution, BlindAI provided an privacy-preversing SDK allowing data scientits to perform secure inference. The simple use of a python client allow to isolates ML model in a CPU protected enclave. Our secure server, deployable on Azure DSC VM, is implemented in Rust, allowing an enforced memory safety.
