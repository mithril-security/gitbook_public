## SGX-ES-PDMS

**SGX-ES-PDMS** is software implementation of an **E**xtensible and **S**ecure **P**ersonal **D**ata **M**anagement **S**ystem (**ES-PDMS**) using Intel **S**oftware **G**uard e**X**tensions, and developped with Open Enclave SDK.<br/>
This implementation is based on the paper "_Personal Data Management Systems: The security and functionality standpoint_" ([References and contributors](references.md)).<br/>
DOI : [10.1016/j.is.2018.09.002](10.1016/j.is.2018.09.002)

Authors : Nicolas Anciaux, Philippe Bonnet, Luc Bouganim, Benjamin Nguyen, Philippe Pucheral, Iulian Sandu-Popa, Guillaume Scerri

Developper contact : [floris.thiant@inria.fr](floris.thiant@inria.fr)

This is an ongoing research project. Various features are under development and there are several known bugs.
